# Spotify Clone - Music Streaming Web App

A high-fidelity clone of the Spotify Web Player, designed with premium aesthetics and functional audio playback using your provided song links.

## 🚀 Key Features
- **Dynamic Tracklist**: All 20+ songs you provided are loaded into a "Recently Played" and "Made For You" grid.
- **Functional Player Bar**:
  - Play/Pause, Next/Previous controls.
  - Interactive Progress Bar (click to seek).
  - Working Volume control.
  - Real-time time formatting.
- **Spotify Aesthetic**:
  - Deep black theme (`#121212`).
  - Spotify Green accents (`#1DB954`).
  - Hover animations, glassmorphism, and smooth transitions.
  - Sidebar with navigation and library sections.
  - Search bar and user profile header.

## 🎨 Design System
- **Colors**: Dark mode by default, utilizing Spotify's official palette.
- **Assets**: Utilizes high-quality Unsplash imagery for album art and Font Awesome for crisp icons.
- **Typography**: Montserrat font for a modern, clean look.

## 🛠️ Performance & SEO
- **Semantic HTML**: Proper use of `<aside>`, `<main>`, `<header>`, and `<footer>`.
- **Fast Loading**: Minimal dependencies, pure JS logic.

## 📁 Project Structure
- `index.html`: Core structure and semantic elements.
- `style.css`: All premium styling and layout definitions.
- `script.js`: Audio playback engine and dynamic UI rendering.

To run the app, simply open `index.html` in any modern web browser.
